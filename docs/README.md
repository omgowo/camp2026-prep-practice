# 文件導覽

這個 repo 的文件分成四種用途。讀者不需要全部讀完。

| 用途 | 文件 | 讀者 |
|---|---|---|
| 入口 | `README.md` | 所有人 |
| 操作 | `tasks/`、`notes/README.md`、`profiles/README.md` | 學員、助教 |
| 執行規範 | `AGENTS.md` | Coding Agent、維護者 |
| 課程與維護 | `docs/` | 講師、助教、未來維護者 |

## 建議閱讀路徑

### 學員

1. `README.md`
2. `tasks/01-create-profile.md`
3. `notes/profile-note.template.md`

### 助教

1. `docs/day1-flow.md`
2. `docs/privacy.md`

### 講師

1. `docs/teaching-method.md`

### 維護者

1. `docs/teaching-method.md`
2. `docs/repo-structure.md`
3. `AGENTS.md`

## 文件維護原則

- 學員文件只放操作步驟和檢查點。
- 講師文件說明設計取捨，不寫成逐字講稿。
- Agent 文件寫規則和邊界，不寫課程背景故事。
- 相同流程只在入口簡述一次，細節交給任務文件。
- 不把私人對談、會議紀錄或未整理推演寫進 repo。
