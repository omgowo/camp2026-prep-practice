# 公開資料與隱私

`profiles/*.json` 可能會被部署成公開網頁。請把它視為公開資料。

## 可以公開

- GitHub 帳號
- 暱稱或顯示名稱
- 一句話介紹
- 1 到 3 個興趣主題
- 想代表自己的 OS、瀏覽器、編輯器陣營

## 不應公開

- 電話、Email、LINE、Discord
- 學校、班級、住址
- 生日、年齡
- 不想公開的社群帳號
- 健康、宗教、政治、家庭背景等敏感資訊
- 任何不希望陌生人看到的內容

## 如果 Agent 產生了不該公開的內容

不要 commit。請 Agent 移除，只保留 profile schema 允許的公開欄位。

可以貼：

```text
請移除 profiles/<我的 GitHub 帳號>.json 裡不適合公開的個人資料，只保留 github、displayName、tagline、interests 和 faction。
```

## GitHub 頭貼

展示頁會使用：

```text
https://github.com/<github>.png
```

如果不想讓目前 GitHub 頭貼出現在展示頁，請先更換頭貼，或詢問助教是否可以使用替代帳號。
