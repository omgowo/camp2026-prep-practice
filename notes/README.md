# notes 資料夾

這裡放你寫給 Coding Agent 看的自然語言草稿。

請從 `profile-note.template.md` 複製一份，改成：

```text
notes/<你的 GitHub 帳號>.md
```

例如：

```text
notes/octocat.md
```

## 填寫原則

- 只寫願意公開的內容。
- 不需要手寫 JSON。
- 作業系統、瀏覽器、編輯器請填你平常最常用，或最想代表自己的陣營。
- 不要因為教室電腦現在使用某個環境，就把它當成你的偏好。

接著依照 `tasks/01-create-profile.md` 請 Coding Agent 建立 profile JSON。
